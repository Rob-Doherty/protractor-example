# protractor-example
Example of Protractor test framework, which uses selenium server to run commands in various supported browsers (chrome, headless chrome, firefox, etc)


## DONE:
- setup of Protractor
    - can use default chrome
    - can override `capabilities` to use firefox
    - can override `capabilities` to use headless chrome via Puppeteer
- basic example = `yarn test`
- conf.js usage
    - can use default selenium server or define separate standalone
